Java Language Extensions

A functional oriented library that helps reduce the code clutter and increase readability. It will help in promoting the usage of immutable entities and programming style.

Create null safe code:

Sequence<String> valueList = API.Option(null)
  .ifPresent(this::process)
  .elseThrow(() -> new NotFoundException());

Create and manipulate sequences:

String reducedString = API.List("first", "second", "third")
  .add("fourth")
  .remove("first")
  .filter(s -> s.length > 5)
  .fold("!", (xs, x) -> xs + x);

Use the Try to delegate exception recovery:

String message = API<String>.Try(this::mayThrowException)
  .recover(x -> "default text")
  .get();

Using the modules in your project

Including the language extensions in your project can be done by adding maven dependencies to your existing project. The project is split in several modules that can be included pending on your needs.

Maven <dependency>
  <groupId>com.com.jongsoft.lang</groupId>
  <artifactId>language</artifactId>
  <version>0.0.5</version>
</dependency> Gradle compile 'com.jongsoft.lang:language:0.0.5'
